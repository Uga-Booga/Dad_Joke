Just a dad joke generator. CLI only for now.

Can be installed on Linux and Windows with .deb or .rpm for linux and portable and installer editions for Windows (Will propably have support for MacOS at some point)

Requirments: Curl

How to use:

Windows:

    Download the portable or the installer edition.
    Unzip the folder for the portable edition or run the installer for the installed edition (Make sure to put it in the system PATH, it is one of the installer questions)
    For the portable edition, run the .exe file (It will open a terminal for 5 seconds and will display the dad joke). For the installed edition, open a terminal sesseion and type "dadjoke". It will display the dad joke for 5 seconds before returning to normal command input.
    Enjoy!!

Linux:

    Download the .deb file for Debian/Ubuntu based systems or the .rpm file for Fedora based systems.
    Open a new terminal session and type the following command: For Debian/Ubuntu based systems: "cd Downloads && sudo apt install dadjoke-1.1.0-Linux.deb". For Fedora based systems: "cd Downloads && sudo dnf install dadjoke-1.1.0-Linux.rpm".
    Type "dadjoke" in your terminal session and enjoy!
